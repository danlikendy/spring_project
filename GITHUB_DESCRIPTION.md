## Full Description (Optional - for README or About section)

**Amusement Park Information System**

A professional web application developed with Spring Boot framework for managing and organizing information about amusement park attractions. The system provides a user-friendly interface with modern design inspired by Apple iOS, featuring smooth animations and intuitive navigation.

**Key Features:**
- Complete CRUD operations (Create, Read, Update, Delete)
- Form validation with error messages
- Responsive design for all devices
- Clean, modern UI with iOS-inspired styling
- H2 in-memory database for easy setup
- Real-time data management

**Technology Stack:**
- Backend: Spring Boot 3.2.0, Spring Data JPA
- Frontend: Thymeleaf, HTML5, CSS3
- Database: H2 (in-memory)
- Build Tool: Maven
- Java Version: 17

**Quick Start:**
```bash
mvn spring-boot:run
```
Access the application at: http://localhost:8080/attractions
